Sehr geehrte Kundin, sehr geehrter Kunde,

das selbstextrahierende Archiv "128iPCI_w2k.exe" entpackt die Treiber standardmäßig in den Ordner 
C:\TerraTec 128iPCI\Win2000.

--------------------------------------------------------------------------------------------------

INSTALLATION DER NEUEN TREIBER

Haben Sie die TerraTec 128iPCI eingebaut, so installiert Windows 2000 zunächst
generische, im Betriebssystem vorhandene Treiber für die 128iPCI. 
Verfahren Sie wie folgt um die Treiber zu aktualisieren.

• Rechts-Klicken Sie auf "Arbeitsplatz".

• Wählen Sie aus dem erscheinden Kontext-Menü "Eigenschaften".

• Wechseln Sie auf den Karteireiter "Hardware".

• Starten Sie den "Geräte-Manager".

• Markieren Sie unter "Sound, video and game controllers" den Eintrag "ESS Solo1 PCI AudioDrive (WDM)"

• Recht-klicken Sie auf den markeirten Eintrag und wählen Sie "Eigenschaften" aus dem Kontextmenü aus.

• Wechseln Sie auf den Karteireiter "Treiber"

• Klicken Sie auf "Treiber aktualiesieren" und klicken Sie "Weiter"

• Wählen Sie "Nach einem passenden Treiber für das Gerät suchen (empfohlen)"
  und klicken Sie auf "Weiter".

• Aktivieren Sie nur die Checkbox "Andere Quelle angeben" und klicken Sie "Weiter".

• Geben Sie den Pfad zu den Treiber-Dateien an (Standard: C:\TerraTec 128iPCI\Win2000) und klicken Sie "Weiter",
  oder wählen sie "Durchsuchen" um per Maus in das entsprechende Verzeichnis zu wechseln.

• Bestätigen Sie erneut mit "Weiter".

• Es erscheint ein Dialog-Fenster das darauf aufmerksam macht, dass der Treiber den Sie installieren
  wollen keine digitale Signatur besitzt. Beantworten Sie die Frage, ob Sie fortfahren wollen mit "JA".

  Es kann sein, dass im weiteren Verlauf der Installation erneut ein solcher Dialog erscheint,
  bleiben Sie ruhig und handeln Sie besonnen und überlegt, indem Sie auf "JA" klicken.
  Windows nimmt nun die Treiberinstallation für Sie vor und quittiert den Vorgang mit einigen
  Fenstern, die Sie sich anschauen können. Mehr sollte allerdings an dieser Stelle nicht passieren.
  Bekommen Sie wider Erwarten doch eine Aufforderung, irgendetwas zu tun, und Sie sind
  sich nicht sicher, ist in der Regel das Drücken der Eingabe-Taste eine erfolgversprechende
  Sache.
  Sollte Windows ein weiteres Mal nach einer Treiberdatei fragen, verweisen Sie bitte wieder auf
  das oben genannte Verzeichnis. Außerdem kann es vorkommen (z.B. für den Fall, dass dies die 
  erste Soundkarteninstallation in Ihrem System ist), dass einige Windows-Erweiterungen mit installiert
  werden müssen. Halten Sie dazu bitte Ihre Windows-CD griffbereit.

• Nach abgeschlossener Installation starten Sie Ihren Rechner bitte neu.

• WICHTIG •
Der Software-Wavetable, der sich auf der 128iPCI CD befindet kann NICHT unter Windows 2000
installiert und verwendet werden!
Bitte benutzen Sie den bereits vorhanden Software Synthesizer von Microsoft, den Sie unter
Multimedia --> MIDI auswählen können!

Dear customer,

the self-extracting archive "128iPCI_w2k.exe" unzipps the driver into the folder 
C:\TerraTec 128iPCI\Win2000 by default.

--------------------------------------------------------------------------------------------------

INSTALLATION OF THE NEW DRIVER

If you have installed the TerraTec 128iPCI, Windows 2000 will first install generic
drivers for the 128iPCI.
Follow the further described steps to update the drivers

• Right-Click on "My Computer".

• Choose "Properties" from the appearing context menu.

• Chnage to the "Hardware" Tab.

• Start the "Device Manager".

• Mark the entry "ESS Solo1 PCI AudioDrive (WDM)" under "Sound, video and game controllers" .

• Right-Click on the marked entry and choose "Properties" from the context menu.

• Change to the "Driver" tab

• Click on the "update Driver" button

• Choose "Search for a suitable driver for my device (recommended)" and click "Next".

• Select the "Specify a location" checkbox and click "Next".

• Type in the path, where the driver files are (default: C:\TerraTec 128iPCI\Win2000) and click "OK",
  or choose "Browse" to change to the appropriate folder.

• Confirm again with "Next".

• It appears a dialog window saying the driver you are going to install has no no digital signature.
  Don't care and proceed with a click on "YES".

  Later in the course of the installation, a window of this kind may appear again, stay calm and
  collected and proceed carefully by clicking "YES".
  Windows now installs the driver for you, documenting the process with several installation
  screens. At this point nothing else should occur. If during this process you are unexpectedly
  prompted to do something and you are unsure how to proceed, it is usually best to just press
  the Enter key.
  If Windows asks for a driver file again, reenter the path to the folder given above. It
  may also be the case that a few other Windows features need to be installed along with the
  card (e.g. if this is the first sound card installation for your system). For this eventuality, 
  please have your Windows CD handy.

 
• After installation succeeded please retsrat your PC to make the changes take effect.

• IMPORTANT •
The software wavetable shipping with the 128iPCI CD can not be installed and used under Windows 2000!
Please use the already available software synthesizer from Microsoft, which can be selected under
Multimedia --> MIDI!
