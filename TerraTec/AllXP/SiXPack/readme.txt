1.0) Installation der neuen Treiber 

1.1) Installation unter Windows 2000

	• Starten Sie die Datei SiXPack_full.EXE.

	• Wählen Sie anschließend "Reboot"

	• Nachdem der Rechner neu gestartet und die neue Hardware durch den Hardwareassistenten
	  von Windows 2000 gefunden wurde, klicken Sie auf "Weiter".

	• Wählen Sie "Nach einem passenden Treiber für das Gerät suchen (empfohlen)"
	  und klicken Sie auf "Weiter".

	• Aktivieren Sie die Checkbox "Andere Quelle angeben" und klicken Sie "Weiter".

	• Geben Sie den Pfad zu den Treiber-Dateien an (Standard: C:\SiXPack5.1+) und 
	  klicken Sie Weiter, oder wählen sie DURCHSUCHEN um per Maus in das entsprechende Verzeichnis 
          zu wechseln.

	• Bestätigen Sie erneut mit "Weiter".

	• Es erscheint ein Dialog-Fenster das darauf aufmerksam macht, dass der Treiber den Sie 
          installieren wollen keine digitale Signatur besitzt. Beantworten Sie die Frage, ob Sie 
          fortfahren wollen mit "JA".

  	Es kann sein, dass im weiteren Verlauf der Installation erneut ein solcher Dialog erscheint,
	bleiben Sie ruhig und handeln Sie besonnen und überlegt, indem Sie auf "JA" klicken.
  	Windows nimmt nun die Treiberinstallation für Sie vor und quittiert den Vorgang mit einigen
  	Fenstern, die Sie sich anschauen können. Mehr sollte allerdings an dieser Stelle nicht passieren.
  	Bekommen Sie wider Erwarten doch eine Aufforderung, irgendetwas zu tun, und Sie sind
  	sich nicht sicher, ist in der Regel das Drücken der Eingabe-Taste eine erfolgversprechende
  	Sache.
	Sollte Windows ein weiteres Mal nach einer Treiberdatei fragen, verweisen Sie bitte wieder auf
	das oben genannte Verzeichnis. Außerdem kann es vorkommen (z.B. für den Fall, dass dies die 
	erste Soundkarteninstallation in Ihrem System ist), dass einige Windows-Erweiterungen mit installiert
	werden müssen. Halten Sie dazu bitte Ihre Windows-CD griffbereit.

	• Nach abgeschlossener Installation ist die SiXPack 5.1+ sofort einsatzbereit.

1.2) Installation unter Windows XP

	• Starten Sie die Datei SiXPack_full.EXE.

	• Wählen Sie anschließend "Reboot"

	• Nachdem der Rechner neu gestartet und die neue Hardware durch den Hardwareassistenten
	  von Windows Me gefunden wurde, wählen Sie "Software automatisch installieren (empfohlen)" 
          und klicken "Weiter". 

	• Bei dem Popup "Treibersignatur" wählen Sie "Installation fortsetzen"

	• Geben Sie den Pfad C:\SiXPack5.1+ ein und klicken Sie auf "Weiter", oder 
	  wählen sie "Durchsuchen" um per Maus in das entsprechende Verzeichnis zu wechseln.

	• Klicken Sie erneut auf "Weiter".

	• Bestätigen Sie den darauf folgenden Dialog mit "Fertig stellen".

1.0) Installation of the new driver

1.1) Installation under Windows 2000

	• Start the SiXPack_full.EXE.

	• Choose "Reboot".

	• After the computer has restarted and the new hardware has been found by the Windows 2000
	  hardware clicken "Next".

	• Choose "Search for a suitable driver for my device (recommended)" and click "Next".

	• Select the "Specify a location" checkbox and click "Next".

	• Type in the path, where the driver files are (default: C:\SiXPack5.1+) and click OK,
	  or choose BROWSE to change to the appropriate folder.

	• Confirm again with "Next".

	• It appear a dialog windows saying the driver you are going to install has no no digital 
	  signature. Don't care and proceed with a click on "YES".

	Later in the course of the installation, a window of this kind may appear again, stay calm and
        collected and proceed carefully by clicking "YES".
  	Windows now installs the driver for you, documenting the process with several installation
  	screens. At this point nothing else should occur. If during this process you are unexpectedly
  	prompted to do something and you are unsure how to proceed, it is usually best to just press
  	the Enter key.
  	If Windows asks for a driver file again, reenter the path to the folder given above. It
  	may also be the case that a few other Windows features need to be installed along with the
  	card (e.g. if this is the first sound card installation for your system). For this eventuality, 
  	please have your Windows CD handy.

 
	• After installation succeeded the SiXPack 5.1+ is immediately ready to go.


1.2) Installation under Windows XP

	• Start the SiXPack_full.EXE program.

	• Choose "Reboot".

	• After the computer restarted and the new hardware was found by the Windows Me
	  hardware assistant choose "Install Hardware automatically (recommended)" and click "Next".

	• At the "Digital signature" Popup chose "Continue Installation"

	• Type in the path, where the driver files are (default: C:\SiXPack5.1+) and 
	  click "Next", or choose "Browse" to change to the appropriate folder.

	• Confirm again with "Next".

	• Click on "Finish" to complete the installation.
